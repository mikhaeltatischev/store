# Store

Сервис для управления товарами (продуктами) с поддержкой пагинации.

## Стек технологий

- **Java 21**
- **Spring Boot 3.x**
- **Spring Data JPA** - работа с базой данных
- **PostgresSQL** - основная БД
- **OpenAPI/Swagger** - документация API
- **JUnit 5 / MockMvc** - тестирование
- **Lombok** - уменьшение шаблонного кода
- **Maven** - сборка проекта

## Архитектура

Проект следует многослойной архитектуре:

- **Controller** - REST endpoints, валидация входных данных
- **Service** - бизнес-логика
- **Repository** - работа с БД через Spring Data JPA
- **Domain** - бизнес-модели с логикой предметной области
- **Entity** - JPA сущности
- **DTO** - объекты для передачи данных
- **Mapper** - преобразование между слоями

## API Endpoints
Описание API можно получить с помощью Swagger

## Модели данных
Схема моделей данных находится в файле Store schema.drawio

## Планирование разработки
Основные задачи на MVP-0, MVP-1, MVP-2 оценены и внесены в файл Store plan.xlsx

Для запуска Development версии проекта необходимо указать 'dev' spring profile в конфигурации запуска -
VM Options: -Dspring.profiles.active=dev
